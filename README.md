# Smart Classroom AI

An AI-powered classroom monitoring and engagement analysis system using YOLOv8, MediaPipe, OpenCV, and DeepSORT for real-time student tracking, attention detection, phone usage monitoring, and classroom analytics.

## Features

* Real-time student detection
* Multi-student tracking
* Attention analysis
* Head-down detection
* Phone detection
* Face landmark tracking
* Classroom analytics dashboard

## Technologies Used

* Python
* OpenCV
* MediaPipe
* YOLOv8
* DeepSORT
* Streamlit

## Run Project

### Start Classroom AI

```bash
python smart_classroom.py
```

### Start Dashboard

```bash
streamlit run dashboard.py
```
